## Overview

This document provides complete, production-ready TypeScript source for every
ElizaOS plugin in the Regen Network agentic tokenomics system. Each plugin
follows ElizaOS conventions:

- **Actions** implement the `Action` interface (`name`, `description`,
  `validate`, `handler`) and represent discrete tasks an agent can perform.
- **Evaluators** implement the `Evaluator` interface (`name`, `description`,
  `validate`, `handler`) and run automatically after each message cycle to
  assess state.
- **Providers** implement the `Provider` interface (`name`, `description`,
  `get`) and supply contextual data to the agent's working memory.
- **Services** extend the `Service` base class for long-running background
  processes.
- **Character definitions** are JSON objects that configure agent personality,
  model provider, and plugin bindings.

All imports come from `@elizaos/core`. The runtime object (`IAgentRuntime`)
provides access to memory, model calls, settings, and inter-agent messaging.

### Prerequisites

```
Node.js >= 20
pnpm >= 9
ElizaOS >= 0.2.0
```

### Quick Start

```bash
# Clone the monorepo
git clone https://github.com/regen-network/agentic-tokenomics.git
cd agentic-tokenomics/packages

# Install all packages
pnpm install

# Build shared types first, then plugins
pnpm --filter plugin-regen-ledger build

# Start ElizaOS with all agents
pnpm start --characters="characters/*.json"
```

## plugin-regen-ledger

The ledger plugin provides MCP client connectivity, a state provider that
injects chain data into agent context, and a health evaluator that monitors
chain liveness.

### LedgerMCPClient

```typescript
// plugin-regen-ledger/src/client.ts

import { IAgentRuntime } from "@elizaos/core";
import {
  MCPServerConfig,
  MCPToolCall,
  MCPToolResult,
  CreditClass,
  Project,
  Proposal,
  Batch,
  Validator,
} from "./types";

export class LedgerMCPClient {
  private config: MCPServerConfig;
  private runtime: IAgentRuntime;

  constructor(runtime: IAgentRuntime) {
    this.runtime = runtime;
    this.config = {
      url: runtime.getSetting("REGEN_MCP_URL") ?? "http://localhost:3100",
      apiKey: runtime.getSetting("REGEN_MCP_API_KEY"),
      timeout: Number(runtime.getSetting("REGEN_MCP_TIMEOUT") ?? "30000"),
      retryAttempts: Number(runtime.getSetting("REGEN_MCP_RETRIES") ?? "3"),
      retryDelay: Number(runtime.getSetting("REGEN_MCP_RETRY_DELAY") ?? "1000"),
    };
  }

  // ── Core transport ──────────────────────────────────────────────────────

  private async callTool(call: MCPToolCall): Promise<MCPToolResult> {
    const startTime = Date.now();
    let lastError: Error | undefined;

    for (let attempt = 0; attempt < this.config.retryAttempts; attempt++) {
      try {
        const controller = new AbortController();
        const timeoutId = setTimeout(
          () => controller.abort(),
          this.config.timeout
        );

        const headers: Record<string, string> = {
          "Content-Type": "application/json",
        };
        if (this.config.apiKey) {
          headers["Authorization"] = `Bearer ${this.config.apiKey}`;
        }

        const response = await fetch(`${this.config.url}/tools/${call.tool}`, {
          method: "POST",
          headers,
          body: JSON.stringify(call.params),
          signal: controller.signal,
        });

        clearTimeout(timeoutId);

        if (!response.ok) {
          throw new Error(
            `MCP call failed: ${response.status} ${response.statusText}`
          );
        }

        const data = await response.json();
        return {
          success: true,
          data,
          latencyMs: Date.now() - startTime,
        };
      } catch (err) {
        lastError = err instanceof Error ? err : new Error(String(err));

        if (attempt < this.config.retryAttempts - 1) {
          await new Promise((resolve) =>
            setTimeout(resolve, this.config.retryDelay * (attempt + 1))
          );
        }
      }
    }

    return {
      success: false,
      error: lastError?.message ?? "Unknown MCP error",
      latencyMs: Date.now() - startTime,
    };
  }

  // ── Ecocredit Module ────────────────────────────────────────────────────

  async listClasses(pagination?: {
    offset: number;
    limit: number;
  }): Promise<CreditClass[]> {
    const result = await this.callTool({
      tool: "regen.ecocredit.v1.Query/Classes",
      params: {
        pagination: pagination ?? { offset: 0, limit: 100 },
      },
    });

    if (!result.success) {
      throw new Error(`Failed to list classes: ${result.error}`);
    }

    const response = result.data as { classes: CreditClass[] };
    return response.classes;
  }

  async listProjects(classId?: string): Promise<Project[]> {
    const tool = classId
      ? "regen.ecocredit.v1.Query/ProjectsByClass"
      : "regen.ecocredit.v1.Query/Projects";

    const result = await this.callTool({
      tool,
      params: classId ? { classId } : {},
    });

    if (!result.success) {
      throw new Error(`Failed to list projects: ${result.error}`);
    }

    const response = result.data as { projects: Project[] };
    return response.projects;
  }

  async getProposal(proposalId: number): Promise<Proposal> {
    const result = await this.callTool({
      tool: "cosmos.gov.v1beta1.Query/Proposal",
      params: { proposalId },
    });

    if (!result.success) {
      throw new Error(`Failed to get proposal ${proposalId}: ${result.error}`);
    }

    return (result.data as { proposal: Proposal }).proposal;
  }

  async listSellOrders(batchDenom?: string): Promise<unknown[]> {
    const tool = batchDenom
      ? "regen.ecocredit.marketplace.v1.Query/SellOrdersByBatch"
      : "regen.ecocredit.marketplace.v1.Query/SellOrders";

    const result = await this.callTool({
      tool,
      params: batchDenom ? { batchDenom } : {},
    });

    if (!result.success) {
      throw new Error(`Failed to list sell orders: ${result.error}`);
    }

    return (result.data as { sellOrders: unknown[] }).sellOrders;
  }

  async listValidators(): Promise<Validator[]> {
    const result = await this.callTool({
      tool: "cosmos.staking.v1beta1.Query/Validators",
      params: { status: "BOND_STATUS_BONDED" },
    });

    if (!result.success) {
      throw new Error(`Failed to list validators: ${result.error}`);
    }

    return (result.data as { validators: Validator[] }).validators;
  }

  async queryModuleState(module: string): Promise<Record<string, unknown>> {
    const result = await this.callTool({
      tool: `${module}/ModuleState`,
      params: {},
    });

    if (!result.success) {
      throw new Error(
        `Failed to query module state for ${module}: ${result.error}`
      );
    }

    return result.data as Record<string, unknown>;
  }
}
```

### LedgerStateProvider

```typescript
// plugin-regen-ledger/src/providers.ts

import {
  Provider,
  IAgentRuntime,
  Memory,
  State,
} from "@elizaos/core";
import { LedgerMCPClient } from "./client";

export const ledgerStateProvider: Provider = {
  name: "LEDGER_STATE",
  description:
    "Provides current Regen Ledger state including credit classes, " +
    "projects, proposals, and validator set to the agent context.",

  get: async (
    runtime: IAgentRuntime,
    message: Memory,
    state?: State
  ): Promise<string> => {
    const client = new LedgerMCPClient(runtime);

    try {
      const [classes, projects, validators] = await Promise.all([
        client.listClasses({ offset: 0, limit: 20 }),
        client.listProjects(),
        client.listValidators(),
      ]);

      const chainState = {
        timestamp: new Date().toISOString(),
        creditClasses: {
          count: classes.length,
          recent: classes.slice(0, 5).map((c) => ({
            id: c.id,
            type: c.creditTypeAbbrev,
            batches: c.numBatches,
            projects: c.numProjects,
          })),
        },
        projects: {
          count: projects.length,
          byStatus: projects.reduce(
            (acc, p) => {
              acc[p.status] = (acc[p.status] || 0) + 1;
              return acc;
            },
            {} as Record<string, number>
          ),
        },
        validators: {
          activeCount: validators.filter(
            (v) => v.status === "BOND_STATUS_BONDED"
          ).length,
          totalStake: validators
            .reduce((sum, v) => sum + BigInt(v.tokens), BigInt(0))
            .toString(),
        },
      };

      return (
        "## Current Regen Ledger State\n\n" +
        "```json\n" +
        JSON.stringify(chainState, null, 2) +
        "\n```\n\n" +
        `Last updated: ${chainState.timestamp}`
      );
    } catch (err) {
      const errorMsg =
        err instanceof Error ? err.message : "Unknown error";
      return `[Ledger State Provider] Error fetching chain state: ${errorMsg}`;
    }
  },
};

### ChainHealthEvaluator

```typescript
// plugin-regen-ledger/src/evaluators.ts

import {
  Evaluator,
  IAgentRuntime,
  Memory,
  State,
} from "@elizaos/core";
import { LedgerMCPClient } from "./client";

export const chainHealthEvaluator: Evaluator = {
  name: "CHAIN_HEALTH",
  description:
    "Evaluates the health of the Regen Ledger by checking block production, " +
    "validator uptime, and module responsiveness. Triggers alerts when " +
    "degradation is detected.",

  validate: async (
    runtime: IAgentRuntime,
    message: Memory,
    state?: State
  ): Promise<boolean> => {
    // Run health check every cycle — always valid
    return true;
  },

  handler: async (
    runtime: IAgentRuntime,
    message: Memory,
    state?: State
  ): Promise<void> => {
    const client = new LedgerMCPClient(runtime);

    try {
      const startTime = Date.now();
      const [ecocreditState, validators] = await Promise.all([
        client.queryModuleState("regen.ecocredit.v1"),
        client.listValidators(),
      ]);
      const latencyMs = Date.now() - startTime;

      // Check MCP responsiveness
      const isResponsive = latencyMs < 5000;

      // Check validator set health
      const bondedValidators = validators.filter(
        (v) => v.status === "BOND_STATUS_BONDED"
      );
      const jailedCount = validators.filter((v) => v.jailed).length;
      const validatorHealthy = jailedCount / validators.length < 0.1;

      // Compute overall health
      const healthy = isResponsive && validatorHealthy;

      if (!healthy) {
        const alerts: string[] = [];
        if (!isResponsive) {
          alerts.push(`MCP latency high: ${latencyMs}ms`);
        }
        if (!validatorHealthy) {
          alerts.push(
            `Validator set degraded: ${jailedCount}/${validators.length} jailed`
          );
        }

        // Store alert in agent memory for downstream actions
        await runtime.messageManager.createMemory({
          userId: message.userId,
          agentId: runtime.agentId,
          roomId: message.roomId,
          content: {
            text: `[CHAIN_HEALTH_ALERT] ${alerts.join("; ")}`,
            metadata: {
              type: "chain_health_alert",
              latencyMs,
              bondedValidators: bondedValidators.length,
              jailedValidators: jailedCount,
              isResponsive,
              validatorHealthy,
              timestamp: new Date().toISOString(),
            },
          },
        });
      }
    } catch (err) {
      const errorMsg =
        err instanceof Error ? err.message : "Unknown error";

      await runtime.messageManager.createMemory({
        userId: message.userId,
        agentId: runtime.agentId,
        roomId: message.roomId,
        content: {
          text: `[CHAIN_HEALTH_ERROR] Failed to evaluate: ${errorMsg}`,
          metadata: {
            type: "chain_health_error",
            error: errorMsg,
            timestamp: new Date().toISOString(),
          },
        },
      });
    }
  },
};
```

## plugin-regen-registry (AGENT-001)

The Registry Analyst monitors credit class creation, methodology submissions,
project registrations, and batch issuances. It uses the OODA
(Observe-Orient-Decide-Act) loop pattern for each analysis workflow.

### analyzeMethodologyAction

```typescript
// plugin-regen-registry/src/actions/analyzeMethodology.ts

import {
  Action,
  IAgentRuntime,
  Memory,
  State,
  HandlerCallback,
  ModelClass,
  generateText,
} from "@elizaos/core";
import { LedgerMCPClient } from "plugin-regen-ledger";
import type {
  MethodologyScore,
  AgentDecision,
  WorkflowExecution,
} from "plugin-regen-ledger";

export const analyzeMethodologyAction: Action = {
  name: "ANALYZE_METHODOLOGY",
  description:
    "Performs a full OODA-loop analysis of a credit class methodology. " +
    "Observes the metadata IRI, orients by assessing completeness and " +
    "scientific rigor, decides on a quality score, and acts by submitting " +
    "the assessment to the registry.",

  validate: async (
    runtime: IAgentRuntime,
    message: Memory,
    state?: State
  ): Promise<boolean> => {
    const text = message.content?.text?.toLowerCase() ?? "";
    return (
      text.includes("methodology") ||
      text.includes("credit class") ||
      text.includes("analyze class")
    );
  },

  handler: async (
    runtime: IAgentRuntime,
    message: Memory,
    state?: State,
    options?: Record<string, unknown>,
    callback?: HandlerCallback
  ): Promise<void> => {
    const client = new LedgerMCPClient(runtime);
    const classId =
      (message.content?.metadata?.classId as string) ??
      extractClassId(message.content?.text ?? "");

    if (!classId) {
      await callback?.({
        text: "I need a credit class ID to analyze. Please provide one like C01 or C02.",
      });
      return;
    }

    const workflow: WorkflowExecution = {
      workflowId: `methodology-${classId}-${Date.now()}`,
      agentId: "AGENT-001",
      phase: "OBSERVE",
      input: { classId },
      output: {},
      startedAt: new Date().toISOString(),
      status: "RUNNING",
    };

    try {
      // ── OBSERVE: Fetch methodology metadata via IRI ─────────────────
      workflow.phase = "OBSERVE";
      await callback?.({
        text: `Observing methodology for credit class ${classId}...`,
      });

      const classes = await client.listClasses();
      const targetClass = classes.find((c) => c.id === classId);

      if (!targetClass) {
        await callback?.({
          text: `Credit class ${classId} not found on chain.`,
        });
        return;
      }

      const metadataIri = targetClass.metadata;
      const projects = await client.listProjects(classId);

      // ── ORIENT: Assess completeness and scientific rigor ────────────
      workflow.phase = "ORIENT";
      await callback?.({
        text: `Orienting: assessing completeness of methodology at ${metadataIri}...`,
      });

      const orientPrompt = `You are a scientific methodology reviewer for ecological credit classes.

Analyze the following credit class:
- Class ID: ${targetClass.id}
- Credit Type: ${targetClass.creditTypeAbbrev}
- Number of Issuers: ${targetClass.issuers.length}
- Number of Projects: ${targetClass.numProjects}
- Number of Batches: ${targetClass.numBatches}
- Metadata IRI: ${metadataIri}

Assess the methodology on these dimensions (0.0 to 1.0):
1. completeness - Are all required methodology sections present?
2. scientificRigor - Is the science sound and peer-reviewable?
3. monitoringQuality - Are MRV requirements clear and achievable?
4. additionalityClarity - Is the additionality argument well-defined?

Also list any red flags.

Respond as JSON:
{
  "completeness": number,
  "scientificRigor": number,
  "monitoringQuality": number,
  "additionalityClarity": number,
  "flags": string[]
}`;

      const orientResponse = await generateText({
        runtime,
        context: orientPrompt,
        modelClass: ModelClass.LARGE,
      });

      let score: MethodologyScore;
      try {
        const parsed = JSON.parse(orientResponse);
        score = {
          completeness: parsed.completeness,
          scientificRigor: parsed.scientificRigor,
          monitoringQuality: parsed.monitoringQuality,
          additionalityClarity: parsed.additionalityClarity,
          overallScore:
            (parsed.completeness +
              parsed.scientificRigor +
              parsed.monitoringQuality +
              parsed.additionalityClarity) /
            4,
          flags: parsed.flags ?? [],
        };
      } catch {
        score = {
          completeness: 0.5,
          scientificRigor: 0.5,
          monitoringQuality: 0.5,
          additionalityClarity: 0.5,
          overallScore: 0.5,
          flags: ["Failed to parse LLM assessment; defaulting to 0.5"],
        };
      }

      // ── DECIDE: Determine outcome based on score thresholds ─────────
      workflow.phase = "DECIDE";

      const decision: AgentDecision = {
        agentId: "AGENT-001",
        action: "ANALYZE_METHODOLOGY",
        outcome:
          score.overallScore >= 0.7
            ? "APPROVE"
            : score.overallScore >= 0.4
              ? "ESCALATE"
              : "REJECT",
        confidence: score.overallScore,
        reasoning: `Methodology overall score: ${score.overallScore.toFixed(2)}. ` +
          `Completeness: ${score.completeness.toFixed(2)}, ` +
          `Scientific rigor: ${score.scientificRigor.toFixed(2)}, ` +
          `Monitoring: ${score.monitoringQuality.toFixed(2)}, ` +
          `Additionality: ${score.additionalityClarity.toFixed(2)}.` +
          (score.flags.length > 0
            ? ` Flags: ${score.flags.join(", ")}`
            : ""),
        evidence: {
          classId,
          metadataIri,
          score,
          projectCount: projects.length,
        },
        timestamp: new Date().toISOString(),
      };

      if (decision.outcome === "ESCALATE") {
        decision.escalation = {
          targetAgentId: "AGENT-002",
          reason: `Methodology score ${score.overallScore.toFixed(2)} is in ambiguous range. Governance review recommended.`,
          priority: "MEDIUM",
          context: { classId, score },
        };
      }

      // ── ACT: Submit assessment and notify ───────────────────────────
      workflow.phase = "ACT";
      workflow.output = { decision, score };
      workflow.completedAt = new Date().toISOString();
      workflow.status = "COMPLETED";

      // Store decision in memory
      await runtime.messageManager.createMemory({
        userId: message.userId,
        agentId: runtime.agentId,
        roomId: message.roomId,
        content: {
          text: `[METHODOLOGY_ASSESSMENT] ${classId}: ${decision.outcome}`,
          metadata: {
            type: "methodology_assessment",
            ...decision,
          },
        },
      });

      const report =
        `## Methodology Analysis: ${classId}\n\n` +
        `**Outcome:** ${decision.outcome}\n` +
        `**Confidence:** ${decision.confidence.toFixed(2)}\n\n` +
        `### Scores\n` +
        `| Dimension | Score |\n` +
        `|-----------|-------|\n` +
        `| Completeness | ${score.completeness.toFixed(2)} |\n` +
        `| Scientific Rigor | ${score.scientificRigor.toFixed(2)} |\n` +
        `| Monitoring Quality | ${score.monitoringQuality.toFixed(2)} |\n` +

          ? `### Flags\n${score.flags.map((f) => `- ${f}`).join("\n")}\n\n`

export const duplicateDetectorEvaluator: Evaluator = {

  description:
    "Detects potential duplicate projects within the same credit class " +
    "by comparing jurisdiction, reference IDs, and metadata similarity.",

  validate: async (
    runtime: IAgentRuntime,
    message: Memory,
    state?: State
  ): Promise<boolean> => {

    state?: State
  ): Promise<void> => {

    try {
      const projects = await client.listProjects();

      // Group by class + jurisdiction to find potential duplicates
      const groups = new Map<string, typeof projects>();
      for (const project of projects) {
        const key = `${project.classId}:${project.jurisdiction}`;
        const group = groups.get(key) ?? [];
        group.push(project);
        groups.set(key, group);
      }

      const duplicateCandidates: Array<{
        classId: string;
        jurisdiction: string;
        projects: string[];
      }> = [];

      for (const [key, group] of groups) {
        if (group.length > 1) {
          const [classId, jurisdiction] = key.split(":");
          duplicateCandidates.push({
            classId,
            jurisdiction,
            projects: group.map((p) => p.id),
          });
        }
      }

      if (duplicateCandidates.length > 0) {
        const summary = duplicateCandidates
          .map(
            (d) =>
              `- Class ${d.classId}, ${d.jurisdiction}: ` +
              `[${d.projects.join(", ")}]`
          )
          .join("\n");

        await runtime.messageManager.createMemory({
          userId: message.userId,
          agentId: runtime.agentId,
          roomId: message.roomId,

            text:
              `[DUPLICATE_DETECTION] ${duplicateCandidates.length} potential ` +
              `duplicate groups found:\n${summary}`,
            metadata: {
              type: "duplicate_detection",
              candidates: duplicateCandidates,
              timestamp: new Date().toISOString(),

          },
        });

  "confidence": number
}`;

      const classifyResponse = await generateText({
        runtime,
        context: classifyPrompt,
        modelClass: ModelClass.LARGE,
      });

      let analysis: ProposalAnalysis;
      try {
        const parsed = JSON.parse(classifyResponse);
        analysis = {
          proposalId: proposal.proposalId,
          classification: parsed.classification,
          impactAssessment: parsed.impactAssessment,
          sentimentSummary: parsed.sentimentSummary,
          precedents: parsed.precedents ?? [],
          recommendation: parsed.recommendation,
          confidence: parsed.confidence,
        };
      } catch {
        analysis = {
          proposalId: proposal.proposalId,
          classification: "unknown",
          impactAssessment: "Unable to assess — LLM parse failure",
          sentimentSummary: "Unable to assess",
          precedents: [],
          recommendation: "DEFER",
          confidence: 0.3,
        };
      }

      const decision: AgentDecision = {
        agentId: "AGENT-002",
        action: "ANALYZE_PROPOSAL",
        outcome: analysis.recommendation as AgentDecision["outcome"],
        confidence: analysis.confidence,
        reasoning:
          `Proposal #${proposalId} classified as "${analysis.classification}". ` +
          `Impact: ${analysis.impactAssessment}. ` +
          `Sentiment: ${analysis.sentimentSummary}.`,
        evidence: { proposalId, analysis, tally: proposal.finalTallyResult },
        timestamp: new Date().toISOString(),
      };

      await runtime.messageManager.createMemory({
        userId: message.userId,
        agentId: runtime.agentId,
        roomId: message.roomId,
        content: {
          text: `[PROPOSAL_ANALYSIS] #${proposalId}: ${decision.outcome}`,
          metadata: { type: "proposal_analysis", ...decision },
        },
      });

      // Calculate tally percentages
      const totalVotes =
        BigInt(proposal.finalTallyResult.yes) +
        BigInt(proposal.finalTallyResult.no) +
        BigInt(proposal.finalTallyResult.abstain) +
        BigInt(proposal.finalTallyResult.noWithVeto);

      const pct = (val: string) =>
        totalVotes > 0n
          ? ((Number(BigInt(val) * 10000n / totalVotes) / 100).toFixed(1) + "%")
          : "N/A";

      const report =
        `## Proposal Analysis: #${proposalId}\n\n` +
        `**Title:** ${proposal.content.title}\n` +
        `**Status:** ${proposal.status}\n` +
        `**Classification:** ${analysis.classification}\n` +
        `**Recommendation:** ${analysis.recommendation} ` +
        `(confidence: ${analysis.confidence.toFixed(2)})\n\n` +
        `### Impact Assessment\n${analysis.impactAssessment}\n\n` +
        `### Community Sentiment\n${analysis.sentimentSummary}\n\n` +
        `### Current Tally\n` +
        `| Vote | Amount | Percentage |\n` +
        `|------|--------|------------|\n` +
        `| Yes | ${proposal.finalTallyResult.yes} | ${pct(proposal.finalTallyResult.yes)} |\n` +
        `| No | ${proposal.finalTallyResult.no} | ${pct(proposal.finalTallyResult.no)} |\n` +
        `| Abstain | ${proposal.finalTallyResult.abstain} | ${pct(proposal.finalTallyResult.abstain)} |\n` +
        `| No w/ Veto | ${proposal.finalTallyResult.noWithVeto} | ${pct(proposal.finalTallyResult.noWithVeto)} |\n\n` +
        (analysis.precedents.length > 0
          ? `### Precedents\n${analysis.precedents.map((p) => `- ${p}`).join("\n")}\n`
          : "");

      await callback?.({ text: report });
    } catch (err) {
      const errorMsg =
        err instanceof Error ? err.message : "Unknown error";

        text: `Failed to analyze proposal #${proposalId}: ${errorMsg}`,

    }

import {
  Action,
  IAgentRuntime,

import { LedgerMCPClient } from "plugin-regen-ledger";

export const trackVotingAction: Action = {
  name: "TRACK_VOTING",
  description:
    "Tracks real-time voting progress on an active governance proposal, " +
    "including tally updates, quorum monitoring, and outcome projection.",

  validate: async (
    runtime: IAgentRuntime,
    message: Memory,
    state?: State
  ): Promise<boolean> => {
    const text = message.content?.text?.toLowerCase() ?? "";
    return (
      text.includes("track vot") ||
      text.includes("voting progress") ||
      text.includes("vote status")
    );
  },

  handler: async (
    runtime: IAgentRuntime,
    message: Memory,
    state?: State,
    options?: Record<string, unknown>,
    callback?: HandlerCallback
  ): Promise<void> => {
    const client = new LedgerMCPClient(runtime);
    const proposalId =
      (message.content?.metadata?.proposalId as number) ??
      extractProposalId(message.content?.text ?? "");

    if (!proposalId) {
      await callback?.({
        text: "Please provide a proposal ID to track (e.g., #42).",
      });
      return;
    }

    try {
      const proposal = await client.getProposal(proposalId);
      const validators = await client.listValidators();

      // Calculate total bonded stake for quorum calculation
      const totalBonded = validators
        .filter((v) => v.status === "BOND_STATUS_BONDED")
        .reduce((sum, v) => sum + BigInt(v.tokens), BigInt(0));

      const tally = proposal.finalTallyResult;
      const totalVoted =
        BigInt(tally.yes) +
        BigInt(tally.no) +
        BigInt(tally.abstain) +
        BigInt(tally.noWithVeto);

      const quorumPct =
        totalBonded > 0n
          ? Number((totalVoted * 10000n) / totalBonded) / 100
          : 0;
      const quorumMet = quorumPct >= 33.4; // Cosmos SDK default quorum

      // Time remaining
      const votingEnd = new Date(proposal.votingEndTime);
      const now = new Date();
      const hoursRemaining = Math.max(
        0,
        (votingEnd.getTime() - now.getTime()) / (1000 * 60 * 60)
      );

      // Project outcome based on current trajectory
      const yesPct =
        totalVoted > 0n
          ? Number((BigInt(tally.yes) * 10000n) / totalVoted) / 100
          : 0;
      const vetoPct =
        totalVoted > 0n
          ? Number((BigInt(tally.noWithVeto) * 10000n) / totalVoted) / 100
          : 0;

      let projectedOutcome: string;
      if (vetoPct > 33.4) {
        projectedOutcome = "REJECTED (Veto threshold exceeded)";
      } else if (yesPct > 50) {
        projectedOutcome = quorumMet
          ? "LIKELY PASS"
          : "PASS (if quorum met)";
      } else {
        projectedOutcome = "LIKELY REJECT";
      }

      const report =
        `## Voting Tracker: Proposal #${proposalId}\n\n` +
        `**Status:** ${proposal.status}\n` +
        `**Time Remaining:** ${hoursRemaining.toFixed(1)} hours\n\n` +
        `### Tally\n` +
        `| Vote | Amount | % of Voted |\n` +
        `|------|--------|------------|\n` +
        `| Yes | ${tally.yes} | ${yesPct.toFixed(1)}% |\n` +
        `| No | ${tally.no} | ${totalVoted > 0n ? (Number((BigInt(tally.no) * 10000n) / totalVoted) / 100).toFixed(1) : "0"}% |\n` +
        `| Abstain | ${tally.abstain} | ${totalVoted > 0n ? (Number((BigInt(tally.abstain) * 10000n) / totalVoted) / 100).toFixed(1) : "0"}% |\n` +
        `| No w/ Veto | ${tally.noWithVeto} | ${vetoPct.toFixed(1)}% |\n\n` +
        `### Quorum\n` +
        `- Participation: ${quorumPct.toFixed(1)}% of bonded stake\n` +
        `- Quorum threshold: 33.4%\n` +
        `- Quorum met: ${quorumMet ? "YES" : "NO"}\n\n` +
        `### Projection\n` +
        `**${projectedOutcome}**`;

      await runtime.messageManager.createMemory({
        userId: message.userId,
        agentId: runtime.agentId,
        roomId: message.roomId,
        content: {
          text: `[VOTING_TRACKER] #${proposalId}: ${projectedOutcome}`,
          metadata: {
            type: "voting_tracker",
            proposalId,
            quorumPct,
            quorumMet,
            yesPct,
            vetoPct,
            hoursRemaining,
            projectedOutcome,
            timestamp: new Date().toISOString(),
          },
        },
      });

      await callback?.({ text: report });
    } catch (err) {
      const errorMsg =
        err instanceof Error ? err.message : "Unknown error";
      await callback?.({
        text: `Failed to track voting for proposal #${proposalId}: ${errorMsg}`,
      });
    }
  },
};

### detectAnomalyAction

```typescript
// plugin-regen-market/src/actions/detectAnomaly.ts

import {
  Action,
  IAgentRuntime,
  Memory,
  State,
  HandlerCallback,
} from "@elizaos/core";
import { LedgerMCPClient } from "plugin-regen-ledger";
import type { MarketAnomaly } from "plugin-regen-ledger";

export const detectAnomalyAction: Action = {
  name: "DETECT_ANOMALY",
  description:
    "Scans the credit marketplace for price anomalies using z-score " +
    "statistical analysis. Detects spikes, crashes, volume surges, " +
    "and liquidity drops across all active sell orders.",

  validate: async (
    runtime: IAgentRuntime,
    message: Memory,
    state?: State
  ): Promise<boolean> => {
    const text = message.content?.text?.toLowerCase() ?? "";
    return (
      text.includes("anomaly") ||
      text.includes("price") ||
      text.includes("market scan") ||
      text.includes("detect")
    );
  },

  handler: async (
    runtime: IAgentRuntime,
    message: Memory,
    state?: State,
    options?: Record<string, unknown>,
    callback?: HandlerCallback
  ): Promise<void> => {
    const client = new LedgerMCPClient(runtime);

    try {
      await callback?.({
        text: "Scanning marketplace for anomalies...",
      });

      const sellOrders = (await client.listSellOrders()) as Array<{
        id: string;
        seller: string;
        batchDenom: string;
        quantity: string;
        askPrice: { denom: string; amount: string };
        disableAutoRetire: boolean;
      }>;

      if (sellOrders.length === 0) {
        await callback?.({
          text: "No active sell orders found on the marketplace.",
        });
        return;
      }

      // Group orders by batch denom and compute statistics
      const ordersByBatch = new Map<string, typeof sellOrders>();
      for (const order of sellOrders) {
        const group = ordersByBatch.get(order.batchDenom) ?? [];
        group.push(order);
        ordersByBatch.set(order.batchDenom, group);
      }

      const anomalies: MarketAnomaly[] = [];

      for (const [batchDenom, orders] of ordersByBatch) {
        if (orders.length < 3) continue; // Need enough data for statistics

        const prices = orders.map((o) => Number(o.askPrice.amount));
        const mean = prices.reduce((a, b) => a + b, 0) / prices.length;
        const stdDev = Math.sqrt(
          prices.reduce((sum, p) => sum + Math.pow(p - mean, 2), 0) /
            prices.length
        );

        if (stdDev === 0) continue; // All same price, no anomaly

        for (const order of orders) {
          const price = Number(order.askPrice.amount);
          const zScore = (price - mean) / stdDev;

          if (Math.abs(zScore) > 2.0) {
            const type: MarketAnomaly["type"] =
              zScore > 0 ? "PRICE_SPIKE" : "PRICE_CRASH";
            const severity: MarketAnomaly["severity"] =
              Math.abs(zScore) > 3.0
                ? "CRITICAL"
                : Math.abs(zScore) > 2.5
                  ? "HIGH"
                  : "MEDIUM";

            anomalies.push({
              type,
              severity,
              zScore: Math.round(zScore * 100) / 100,
              affectedAsset: batchDenom,
              detectedAt: new Date().toISOString(),
              description:
                `Order ${order.id} priced at ${price} ${order.askPrice.denom} ` +
                `(mean: ${mean.toFixed(2)}, z-score: ${zScore.toFixed(2)})`,
            });
          }
        }

        // Check for volume anomalies
        const quantities = orders.map((o) => Number(o.quantity));
        const qMean =
          quantities.reduce((a, b) => a + b, 0) / quantities.length;
        const qStdDev = Math.sqrt(
          quantities.reduce((sum, q) => sum + Math.pow(q - qMean, 2), 0) /
            quantities.length
        );

        if (qStdDev > 0) {
          for (const order of orders) {
            const qty = Number(order.quantity);
            const qzScore = (qty - qMean) / qStdDev;

            if (qzScore > 2.5) {
              anomalies.push({
                type: "VOLUME_SURGE",
                severity: qzScore > 3.0 ? "HIGH" : "MEDIUM",
                zScore: Math.round(qzScore * 100) / 100,
                affectedAsset: batchDenom,
                detectedAt: new Date().toISOString(),
                description:
                  `Order ${order.id} quantity ${qty} ` +
                  `(mean: ${qMean.toFixed(2)}, z-score: ${qzScore.toFixed(2)})`,
              });
            }
          }
        }
      }

      // Store anomalies in memory
      if (anomalies.length > 0) {
        await runtime.messageManager.createMemory({
          userId: message.userId,
          agentId: runtime.agentId,
          roomId: message.roomId,
          content: {
            text: `[MARKET_ANOMALIES] Detected ${anomalies.length} anomalies`,
            metadata: {
              type: "market_anomalies",
              anomalies,
              timestamp: new Date().toISOString(),
            },
          },
        });
      }

      const report =
        `## Market Anomaly Scan\n\n` +
        `**Total Orders Scanned:** ${sellOrders.length}\n` +
        `**Batch Denoms Analyzed:** ${ordersByBatch.size}\n` +
        `**Anomalies Detected:** ${anomalies.length}\n\n` +
        (anomalies.length > 0
          ? `### Anomalies\n` +
            `| Type | Severity | Asset | Z-Score | Description |\n` +
            `|------|----------|-------|---------|-------------|\n` +
            anomalies
              .sort((a, b) => Math.abs(b.zScore) - Math.abs(a.zScore))
              .map(
                (a) =>
                  `| ${a.type} | ${a.severity} | ${a.affectedAsset} | ${a.zScore} | ${a.description} |`
              )
              .join("\n") +

        existing.totalQuantity += qty;
        existing.minPrice = Math.min(existing.minPrice, price);
        existing.maxPrice = Math.max(existing.maxPrice, price);
        existing.prices.push(price);
        existing.sellers.add(order.seller);

        batchStats.set(order.batchDenom, existing);

  description:
    "Continuously monitors marketplace prices for statistical anomalies " +
    "using z-score analysis. Alerts when prices deviate significantly " +
    "from batch-level norms.",

  validate: async (
    runtime: IAgentRuntime,
    message: Memory,
    state?: State
  ): Promise<boolean> => {
    // Always run — background monitoring
    return true;
  },

  handler: async (
    runtime: IAgentRuntime,
    message: Memory,
    state?: State
  ): Promise<void> => {
    const client = new LedgerMCPClient(runtime);

    try {
      const sellOrders = (await client.listSellOrders()) as Array<{
        id: string;
        batchDenom: string;
        quantity: string;
        askPrice: { denom: string; amount: string };
      }>;

      if (sellOrders.length < 5) return; // Not enough data

      // Global price statistics
      const allPrices = sellOrders.map((o) => Number(o.askPrice.amount));
      const globalMean =
        allPrices.reduce((a, b) => a + b, 0) / allPrices.length;
      const globalStdDev = Math.sqrt(
        allPrices.reduce(
          (sum, p) => sum + Math.pow(p - globalMean, 2),
          0
        ) / allPrices.length
      );

      if (globalStdDev === 0) return;

      const criticalAnomalies: MarketAnomaly[] = [];

      for (const order of sellOrders) {
        const price = Number(order.askPrice.amount);
        const zScore = (price - globalMean) / globalStdDev;

        if (Math.abs(zScore) > 3.0) {
          criticalAnomalies.push({
            type: zScore > 0 ? "PRICE_SPIKE" : "PRICE_CRASH",
            severity: "CRITICAL",
            zScore: Math.round(zScore * 100) / 100,
            affectedAsset: order.batchDenom,
            detectedAt: new Date().toISOString(),
            description: `Order ${order.id}: ${price} (global mean: ${globalMean.toFixed(2)}, z: ${zScore.toFixed(2)})`,
          });
        }
      }

      if (criticalAnomalies.length > 0) {
        await runtime.messageManager.createMemory({
          userId: message.userId,
          agentId: runtime.agentId,
          roomId: message.roomId,
          content: {
            text:
              `[PRICE_ANOMALY_ALERT] ${criticalAnomalies.length} critical ` +
              `price anomalies detected`,
            metadata: {
              type: "price_anomaly_alert",
              anomalies: criticalAnomalies,
              globalMean,
              globalStdDev,
              timestamp: new Date().toISOString(),
            },
          },
        });
      }
    } catch (err) {

import type { ValidatorAssessment, AgentDecision } from "plugin-regen-ledger";

export const assessValidatorAction: Action = {
  name: "ASSESS_VALIDATOR",
  description:
    "Performs a comprehensive assessment of a validator's performance " +
    "including uptime, governance participation, commission rates, and " +
    "computes a composite score.",

  validate: async (
    runtime: IAgentRuntime,
    message: Memory,

    options?: Record<string, unknown>,

  ): Promise<void> => {
    const client = new LedgerMCPClient(runtime);
    const moniker =
      (message.content?.metadata?.moniker as string) ??

    try {
      const validators = await client.listValidators();

        ? validators.find(

      for (const validator of toAssess) {
        // ── Uptime Score ──────────────────────────────────────────────
        // Based on missed blocks if available, otherwise use jailed status
        const uptimeScore = validator.uptimePercentage
          ? validator.uptimePercentage / 100
          : validator.jailed
            ? 0.0

        // ── Governance Participation Score ─────────────────────────────
        const governanceScore = validator.governanceParticipation
          ? validator.governanceParticipation / 100
          : 0.5; // Default unknown

        // ── Commission Penalty ────────────────────────────────────────
        // High commission rates reduce composite score
        const commissionRate = Number(validator.commission.rate);
        const commissionPenalty =
          commissionRate > 0.2
            ? 0.1
            : commissionRate > 0.1
              ? 0.05
              : 0;

        // ── Composite Score ───────────────────────────────────────────
        const compositeScore = Math.max(
          0,
          Math.min(
            1,
            uptimeScore * 0.5 +
              governanceScore * 0.3 +
              (1 - commissionPenalty) * 0.2
          )
        );

        // ── Risk Level ────────────────────────────────────────────────
        const riskLevel: ValidatorAssessment["riskLevel"] =
          compositeScore >= 0.8

          ? `- ${lowSelfDelegation.length} validators with low minimum self-delegation\n`
          : "") +
        (top10Share > 67
          ? `- CRITICAL: Top 10 validators control ${top10Share.toFixed(1)}% (2/3+ of stake)\n`
          : "");

      await runtime.messageManager.createMemory({
        userId: message.userId,
        agentId: runtime.agentId,

### assessDecentralizationAction

```typescript
// plugin-regen-validator/src/actions/assessDecentralization.ts

import {
  Action,
  IAgentRuntime,
  Memory,
  State,
  HandlerCallback,
} from "@elizaos/core";
import { LedgerMCPClient } from "plugin-regen-ledger";
import type { DecentralizationMetrics } from "plugin-regen-ledger";

export const assessDecentralizationAction: Action = {
  name: "ASSESS_DECENTRALIZATION",
  description:
    "Computes network decentralization metrics including Nakamoto " +
    "coefficient, Gini coefficient, and Herfindahl-Hirschman Index " +
    "for the validator set.",

  validate: async (
    runtime: IAgentRuntime,
    message: Memory,
    state?: State
  ): Promise<boolean> => {
    const text = message.content?.text?.toLowerCase() ?? "";
    return (
      text.includes("decentraliz") ||
      text.includes("nakamoto") ||
      text.includes("gini") ||
      text.includes("network health")
    );
  },

  handler: async (
    runtime: IAgentRuntime,
    message: Memory,
    state?: State,
    options?: Record<string, unknown>,
    callback?: HandlerCallback
  ): Promise<void> => {
    const client = new LedgerMCPClient(runtime);

    try {
      const validators = await client.listValidators();
      const bondedValidators = validators.filter(
        (v) => v.status === "BOND_STATUS_BONDED"
      );

      if (bondedValidators.length === 0) {
        await callback?.({
          text: "No bonded validators found. Cannot assess decentralization.",
        });
        return;
      }

      const stakes = bondedValidators
        .map((v) => Number(v.tokens))
        .sort((a, b) => b - a); // Descending

      const totalStake = stakes.reduce((sum, s) => sum + s, 0);

      if (totalStake === 0) {
        await callback?.({
          text: "Total stake is zero. Cannot compute metrics.",
        });
        return;
      }

      // ── Nakamoto Coefficient ────────────────────────────────────────
      // Minimum number of validators to reach 33.4% of stake (1/3+)
      let cumulativeStake = 0;
      let nakamotoCoefficient = 0;
      for (const stake of stakes) {
        cumulativeStake += stake;
        nakamotoCoefficient++;
        if (cumulativeStake / totalStake >= 0.334) break;
      }

      // ── Gini Coefficient ────────────────────────────────────────────
      const n = stakes.length;
      const sortedAsc = [...stakes].sort((a, b) => a - b);
      let giniNumerator = 0;
      for (let i = 0; i < n; i++) {
        giniNumerator += (2 * (i + 1) - n - 1) * sortedAsc[i];
      }
      const giniCoefficient =
        n > 1
          ? giniNumerator / (n * totalStake)
          : 0;

      // ── Herfindahl-Hirschman Index ──────────────────────────────────
      const shares = stakes.map((s) => s / totalStake);
      const herfindahlIndex = shares.reduce(
        (sum, s) => sum + Math.pow(s, 2),
        0
      );

      // ── Top 10 stake share ──────────────────────────────────────────
      const top10StakeShare =
        stakes.slice(0, 10).reduce((sum, s) => sum + s, 0) / totalStake;

      // ── Assessment ──────────────────────────────────────────────────
      let assessment: string;
      if (nakamotoCoefficient >= 10 && giniCoefficient < 0.5) {
        assessment =
          "HEALTHY: The network has strong decentralization with a high " +
          "Nakamoto coefficient and moderate stake distribution.";
      } else if (nakamotoCoefficient >= 5) {
        assessment =
          "MODERATE: The network has acceptable decentralization but could " +
          "benefit from more balanced stake distribution.";
      } else {
        assessment =
          "CONCERNING: The network has low decentralization. A small " +
          "number of validators could halt or censor the chain.";
      }

      const metrics: DecentralizationMetrics = {
        nakamotoCoefficient,
        giniCoefficient: Math.round(giniCoefficient * 1000) / 1000,
        herfindahlIndex: Math.round(herfindahlIndex * 10000) / 10000,
        top10StakeShare: Math.round(top10StakeShare * 1000) / 10,
        activeValidatorCount: bondedValidators.length,
        assessment,
      };

      await runtime.messageManager.createMemory({
        userId: message.userId,
        agentId: runtime.agentId,
        roomId: message.roomId,
        content: {
          text: `[DECENTRALIZATION] Nakamoto=${metrics.nakamotoCoefficient}, Gini=${metrics.giniCoefficient}`,
          metadata: {
            type: "decentralization_metrics",
            ...metrics,
            timestamp: new Date().toISOString(),
          },
        },
      });

      const report =
        `## Network Decentralization Assessment\n\n` +
        `### Metrics\n` +
        `| Metric | Value | Interpretation |\n` +
        `|--------|-------|----------------|\n` +
        `| Nakamoto Coefficient | ${metrics.nakamotoCoefficient} | ${metrics.nakamotoCoefficient >= 10 ? "Strong" : metrics.nakamotoCoefficient >= 5 ? "Moderate" : "Weak"} |\n` +
        `| Gini Coefficient | ${metrics.giniCoefficient.toFixed(3)} | ${metrics.giniCoefficient < 0.4 ? "Equitable" : metrics.giniCoefficient < 0.6 ? "Moderate inequality" : "High inequality"} |\n` +
        `| Herfindahl Index | ${metrics.herfindahlIndex.toFixed(4)} | ${metrics.herfindahlIndex < 0.1 ? "Competitive" : metrics.herfindahlIndex < 0.25 ? "Moderate" : "Concentrated"} |\n` +
        `| Top 10 Stake Share | ${metrics.top10StakeShare.toFixed(1)}% | ${metrics.top10StakeShare < 50 ? "Healthy" : "Concentrated"} |\n` +
        `| Active Validators | ${metrics.activeValidatorCount} | - |\n\n` +
        `### Assessment\n\n${metrics.assessment}\n\n` +
        `### Methodology\n` +
        `- **Nakamoto Coefficient**: Minimum validators to reach 33.4% of stake\n` +
        `- **Gini Coefficient**: 0 = perfect equality, 1 = total inequality\n` +
        `- **Herfindahl Index**: Sum of squared market shares; <0.1 = competitive\n` +
        `- **Top 10 Share**: Stake concentration in the ten largest validators`;

      await callback?.({ text: report });
    } catch (err) {
      const errorMsg =
        err instanceof Error ? err.message : "Unknown error";
      await callback?.({
        text: `Failed to assess decentralization: ${errorMsg}`,
      });
    }
  },

```typescript
// plugin-regen-validator/src/evaluators/performanceThreshold.ts

import {
  Evaluator,
  IAgentRuntime,
  Memory,
  State,
} from "@elizaos/core";
import { LedgerMCPClient } from "plugin-regen-ledger";

export const performanceThresholdEvaluator: Evaluator = {
  name: "PERFORMANCE_THRESHOLD",
  description:
    "Monitors validator performance and alerts when validators fall " +
    "below acceptable thresholds for uptime, missed blocks, or " +
    "become jailed.",

  validate: async (
    runtime: IAgentRuntime,
    message: Memory,
    state?: State
  ): Promise<boolean> => {
    // Always run as background monitor
    return true;
  },

  handler: async (
    runtime: IAgentRuntime,
    message: Memory,
    state?: State
  ): Promise<void> => {
    const client = new LedgerMCPClient(runtime);

    try {
      const validators = await client.listValidators();

      const alerts: Array<{
        moniker: string;
        address: string;
        issue: string;
        severity: string;
      }> = [];

      for (const validator of validators) {
        // Check for jailed validators
        if (validator.jailed) {
          alerts.push({
            moniker: validator.description.moniker,
            address: validator.operatorAddress,
            issue: "Validator is JAILED",
            severity: "CRITICAL",
          });
        }

        // Check for high missed blocks
        if (
          validator.missedBlocks !== undefined &&
          validator.missedBlocks > 100
        ) {
          alerts.push({
            moniker: validator.description.moniker,
            address: validator.operatorAddress,
            issue: `Missed ${validator.missedBlocks} blocks`,
            severity: validator.missedBlocks > 500 ? "HIGH" : "MEDIUM",
          });
        }

        // Check for low uptime
        if (
          validator.uptimePercentage !== undefined &&
          validator.uptimePercentage < 95
        ) {
          alerts.push({
            moniker: validator.description.moniker,
            address: validator.operatorAddress,
            issue: `Uptime ${validator.uptimePercentage.toFixed(1)}% (below 95% threshold)`,
            severity:
              validator.uptimePercentage < 90 ? "HIGH" : "MEDIUM",
          });
        }

        // Check for excessively high commission
        const commissionRate = Number(validator.commission.rate);
        if (commissionRate > 0.5) {
          alerts.push({
            moniker: validator.description.moniker,
            address: validator.operatorAddress,
            issue: `Commission rate ${(commissionRate * 100).toFixed(0)}% exceeds 50%`,
            severity: "MEDIUM",
          });
        }

        // Check for unbonding validators
        if (validator.status === "BOND_STATUS_UNBONDING") {
          alerts.push({
            moniker: validator.description.moniker,
            address: validator.operatorAddress,
            issue: "Validator is UNBONDING",
            severity: "HIGH",
          });
        }
      }

      if (alerts.length > 0) {
        const criticalCount = alerts.filter(
          (a) => a.severity === "CRITICAL"
        ).length;
        const highCount = alerts.filter(
          (a) => a.severity === "HIGH"
        ).length;

        const summary = alerts
          .sort((a, b) => {
            const order = { CRITICAL: 0, HIGH: 1, MEDIUM: 2 };
            return (
              (order[a.severity as keyof typeof order] ?? 3) -
              (order[b.severity as keyof typeof order] ?? 3)
            );
          })
          .map(
            (a) => `- [${a.severity}] ${a.moniker}: ${a.issue}`
          )
          .join("\n");

        await runtime.messageManager.createMemory({
          userId: message.userId,
          agentId: runtime.agentId,
          roomId: message.roomId,
          content: {
            text:
              `[VALIDATOR_PERFORMANCE_ALERT] ` +
              `${criticalCount} critical, ${highCount} high alerts:\n${summary}`,
            metadata: {
              type: "validator_performance_alert",
              alerts,
              criticalCount,
              highCount,
              timestamp: new Date().toISOString(),
            },
          },
        });
      }
    } catch (err) {
      console.error(
        "[PERFORMANCE_THRESHOLD] Error:",
        err instanceof Error ? err.message : err
      );
    }
  },
};

### Startup Sequence

```typescript
// src/main.ts

import { AgentRuntime, loadCharacters, startServer } from "@elizaos/core";
import config from "../elizaos.config";

async function main(): Promise<void> {
  console.log("Starting Regen Network Agent System...");

  // 1. Load all character definitions
  const characters = await loadCharacters(config.characters);
  console.log(`Loaded ${characters.length} agent characters`);

  // 2. Initialize runtimes for each agent
  const runtimes: AgentRuntime[] = [];

  for (const character of characters) {
    const runtime = new AgentRuntime({
      character,
      modelProvider: config.modelProvider,
      databaseUrl: config.database.url,
      plugins: config.plugins,
    });

    await runtime.initialize();
    runtimes.push(runtime);
    console.log(`Initialized agent: ${character.name}`);
  }

  // 3. Start the HTTP server for all agents
  const server = await startServer({
    runtimes,
    port: Number(process.env.PORT ?? "3000"),
  });

  console.log(`Server started on port ${server.port}`);
  console.log("Active agents:");
  for (const runtime of runtimes) {
    console.log(`  - ${runtime.character.name} (${runtime.agentId})`);
  }

  // 4. Graceful shutdown
  process.on("SIGINT", async () => {
    console.log("Shutting down agents...");
    for (const runtime of runtimes) {
      await runtime.stop();
    }
    process.exit(0);
  });

  process.on("SIGTERM", async () => {
    console.log("Received SIGTERM. Shutting down...");
    for (const runtime of runtimes) {
      await runtime.stop();
    }
    process.exit(0);
  });
}

main().catch((err) => {
  console.error("Fatal error:", err);
  process.exit(1);
});

### Design Patterns

- **OODA Loop**: AGENT-001 uses Observe-Orient-Decide-Act for methodology
  analysis, with clear phase tracking in `WorkflowExecution`.
- **Z-Score Anomaly Detection**: AGENT-003 uses statistical z-scores for
  robust anomaly detection rather than simple threshold alerts.
- **Composite Scoring**: AGENT-004 computes weighted composite scores from
  multiple performance dimensions.
- **Decentralization Indices**: Nakamoto coefficient, Gini coefficient, and
  Herfindahl-Hirschman Index provide institutional-grade metrics.
- **Escalation**: Agents can escalate decisions to peer agents when confidence
  is in an ambiguous range.
- **Memory Persistence**: All decisions, assessments, and alerts are stored via
  `runtime.messageManager.createMemory()` for auditability.

### Key Interfaces

All plugins use `@elizaos/core` types consistently:

- `Action`: `{ name, description, validate, handler }`
- `Evaluator`: `{ name, description, validate, handler }`
- `Provider`: `{ name, description, get }`
- `Plugin`: `{ name, description, actions, evaluators, providers, services }`
- Runtime types: `IAgentRuntime`, `Memory`, `State`, `HandlerCallback`
- Model generation: `generateText()` with `ModelClass.SMALL | LARGE`
