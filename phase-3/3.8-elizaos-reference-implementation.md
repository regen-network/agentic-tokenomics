# Phase 3.8: ElizaOS Reference Implementation

## 1. Overview

This document provides a **complete, working TypeScript reference implementation** for all four Regen Network autonomous agents built on the ElizaOS runtime. While Phase 3.2 defined the agent architecture with pseudocode-level TypeScript, this document delivers production-grade implementations that an engineer can follow to build the actual plugins.

### How to Use This Document

1. **Set up the monorepo** using the package structure in Section 2
2. **Copy the shared types** from Section 3 into `packages/shared/src/types.ts`
3. **Implement each plugin** by following Sections 4-8 in order -- `plugin-regen-ledger` and `plugin-regen-koi` are dependencies for all agent plugins
4. **Configure the runtime** using Section 9 to wire everything together
5. **Deploy characters** using the JSON definitions embedded in each agent section

### Prerequisites

- Node.js 20+ with TypeScript 5.4+
- ElizaOS runtime (`@elizaos/core` latest)
- Access to Regen Ledger MCP server (HTTP endpoint)
- Access to Regen KOI MCP server (HTTP endpoint)
- PostgreSQL 15+ with pgvector extension (for agent memory)
- Anthropic API key (Claude claude-sonnet-4-20250514 or later)

## 4. plugin-regen-ledger -- Complete Implementation

The Ledger plugin provides the MCP client, state provider, and chain health evaluator used by every agent.

### LedgerMCPClient

```typescript
// packages/plugin-regen-ledger/src/client.ts

import type {
  LedgerMCPConfig,
  CreditClass,
  Project,
  CreditBatch,
  SellOrder,
  Proposal,
  Validator,
  Retirement,
  SigningInfo,
  Coin,
} from "@regen/shared";

interface MCPToolResponse<T> {
  success: boolean;
  data: T;
  error?: string;
}

interface PaginatedResponse<T> {
  items: T[];
  pagination: {
    total: number;
    nextKey: string | null;
  };
}

export class LedgerMCPClient {
  private readonly baseUrl: string;
  private readonly apiKey: string;
  private readonly timeoutMs: number;
  private readonly retryAttempts: number;
  private readonly retryDelayMs: number;
  private readonly chainId: string;

  constructor(config: LedgerMCPConfig) {
    this.baseUrl = config.baseUrl.replace(/\/$/, "");
    this.apiKey = config.apiKey;
    this.timeoutMs = config.timeoutMs ?? 30_000;
    this.retryAttempts = config.retryAttempts ?? 3;
    this.retryDelayMs = config.retryDelayMs ?? 1_000;
    this.chainId = config.chainId ?? "regen-1";
  }

  // ── Ecocredit Module ──────────────────────────

  async listClasses(params: {
    limit?: number;
    offset?: number;
  }): Promise<PaginatedResponse<CreditClass>> {
    return this.call("list_classes", {
      limit: params.limit ?? 50,
      offset: params.offset ?? 0,
    });
  }

  async getClass(classId: string): Promise<CreditClass> {
    return this.call("get_class", { class_id: classId });
  }

  async listProjects(params: {
    classId?: string;
    limit?: number;
    offset?: number;
  }): Promise<PaginatedResponse<Project>> {
    return this.call("list_projects", {
      class_id: params.classId,
      limit: params.limit ?? 50,
      offset: params.offset ?? 0,
    });
  }

  async getProject(projectId: string): Promise<Project> {
    return this.call("get_project", { project_id: projectId });
  }

  async listBatches(params: {
    projectId?: string;
    limit?: number;
    offset?: number;
  }): Promise<PaginatedResponse<CreditBatch>> {
    return this.call("list_batches", {
      project_id: params.projectId,
      limit: params.limit ?? 50,
      offset: params.offset ?? 0,
    });
  }

  async getBatch(batchDenom: string): Promise<CreditBatch> {
    return this.call("get_batch", { batch_denom: batchDenom });
  }

  // ── Marketplace Module ────────────────────────

  async listSellOrders(params: {
    batchDenom?: string;
    seller?: string;
    limit?: number;
    page?: number;
  }): Promise<PaginatedResponse<SellOrder>> {
    return this.call("list_sell_orders", {
      batch_denom: params.batchDenom,
      seller: params.seller,
      limit: params.limit ?? 50,
      page: params.page ?? 1,
    });
  }

  async listRetirements(params: {
    batchDenom?: string;
    owner?: string;
    limit?: number;
    offset?: number;
  }): Promise<PaginatedResponse<Retirement>> {
    return this.call("list_retirements", {
      batch_denom: params.batchDenom,
      owner: params.owner,
      limit: params.limit ?? 50,
      offset: params.offset ?? 0,
    });
  }

  // ── Governance Module ─────────────────────────

  async getProposal(proposalId: string | number): Promise<Proposal> {
    return this.call("get_governance_proposal", {
      proposal_id: String(proposalId),
    });

      try {
        const controller = new AbortController();
        const timeoutId = setTimeout(() => controller.abort(), this.timeoutMs);

        const response = await fetch(`${this.baseUrl}/mcp/tools/${tool}`, {
          method: "POST",
          headers: {
            "Content-Type": "application/json",
            Authorization: `Bearer ${this.apiKey}`,
            "X-Chain-Id": this.chainId,
          },
          body: JSON.stringify(params),
          signal: controller.signal,
        });

        clearTimeout(timeoutId);

        if (!response.ok) {
          const errorBody = await response.text().catch(() => "");
          throw new Error(
            `MCP tool "${tool}" failed with status ${response.status}: ${response.statusText}. ${errorBody}`
          );
        }

        const result: MCPToolResponse<T> = await response.json();

        if (!result.success && result.error) {
          throw new Error(`MCP tool "${tool}" returned error: ${result.error}`);
        }

        return result.data;
      } catch (error) {
        lastError = error instanceof Error ? error : new Error(String(error));

        if (error instanceof DOMException && error.name === "AbortError") {
          lastError = new Error(`MCP tool "${tool}" timed out after ${this.timeoutMs}ms`);
        }

        // Do not retry on 4xx client errors
        if (lastError.message.includes("status 4")) {
          throw lastError;
        }

        if (attempt < this.retryAttempts - 1) {
          await this.sleep(this.retryDelayMs * Math.pow(2, attempt));
        }
      }
    }

    throw lastError ?? new Error(`MCP tool "${tool}" failed after ${this.retryAttempts} attempts`);
  }

  private sleep(ms: number): Promise<void> {
    return new Promise((resolve) => setTimeout(resolve, ms));
  }
}
```

```typescript
function assessComponent(
  documents: SearchResult[],
  keywords: string[]
): ComponentAssessment {
  const evidence: string[] = [];
  const gaps: string[] = [];
  let score = 0;

  for (const doc of documents) {
    const content = (doc.content ?? "").toLowerCase();
    const title = (doc.title ?? "").toLowerCase();
    const combined = `${title} ${content}`;

    for (const keyword of keywords) {
      if (combined.includes(keyword.toLowerCase())) {
        evidence.push(`"${keyword}" found in "${doc.title}" (${doc.uri})`);
        score += 1 / keywords.length;
      }
    }
  }

  if (evidence.length === 0) {
    gaps.push(`No documentation found covering: ${keywords.join(", ")}`);
  } else {
    const missingKeywords = keywords.filter(
      (kw) =>
        !evidence.some((e) => e.toLowerCase().includes(kw.toLowerCase()))
    );
    for (const missing of missingKeywords) {
      gaps.push(`Missing coverage for: ${missing}`);
    }
  }

  return {
    found: evidence.length > 0,
    score: Math.min(score, 1.0),
    evidence,
    gaps,
  };
}
```

```typescript
function validateBoundary(metadata: string): {
  valid: boolean;
  issues: string[];
} {
  const issues: string[] = [];

    if (!parsed.projectLocation && !parsed.boundary && !parsed.coordinates) {
      issues.push("No geographic boundary or location data in metadata");
      return { valid: false, issues };
    }

    if (parsed.boundary) {
      if (parsed.boundary.type !== "Polygon" && parsed.boundary.type !== "MultiPolygon") {
        issues.push(`Boundary type "${parsed.boundary.type}" is not a valid polygon`);
      }
      if (!parsed.boundary.coordinates || parsed.boundary.coordinates.length === 0) {
        issues.push("Boundary polygon has no coordinates");
      }
    }

    if (parsed.projectSize) {
      const sizeHa = parseFloat(parsed.projectSize);
      if (isNaN(sizeHa) || sizeHa <= 0) {
        issues.push("Project size is not a valid positive number");
      } else if (sizeHa > 1_000_000) {
        issues.push(`Project size (${sizeHa.toLocaleString()} ha) is unusually large -- verify boundary`);
      }
    }

    return { valid: issues.length === 0, issues };

function assessBaselinePlausibility(
```

```typescript
function validateQuantity(
  batch: CreditBatch,
  historicalBatches: CreditBatch[]
): { valid: boolean; issues: ValidationIssue[] } {
  const issues: ValidationIssue[] = [];
  const quantity = parseFloat(batch.totalAmount);

  if (isNaN(quantity) || quantity <= 0) {
    issues.push({
      severity: "error",
      component: "quantity",
      message: `Batch total amount "${batch.totalAmount}" is not a valid positive number`,
      evidence: null,
    });
    return { valid: false, issues };
  }

  // Check against historical issuance rates
  if (historicalBatches.length > 0) {
    const historicalAmounts = historicalBatches.map((b) =>
      parseFloat(b.totalAmount)
    );
    const avgHistorical =
      historicalAmounts.reduce((sum, a) => sum + a, 0) / historicalAmounts.length;
    const stdDev = Math.sqrt(
      historicalAmounts.reduce(
        (sum, a) => sum + Math.pow(a - avgHistorical, 2),
        0
      ) / historicalAmounts.length
    );

    const zScore =
      stdDev > 0 ? (quantity - avgHistorical) / stdDev : 0;

    if (Math.abs(zScore) > 3) {
      issues.push({
        severity: "error",
        component: "quantity",
        message: `Batch quantity (${quantity.toLocaleString()}) deviates ${zScore.toFixed(1)} standard deviations from historical average (${avgHistorical.toLocaleString()})`,
        evidence: `z-score: ${zScore.toFixed(2)}, historical avg: ${avgHistorical.toFixed(0)}, std: ${stdDev.toFixed(0)}`,
      });
    } else if (Math.abs(zScore) > 2) {
      issues.push({
        severity: "warning",
        component: "quantity",
        message: `Batch quantity is ${zScore.toFixed(1)} standard deviations from the historical average`,
        evidence: `z-score: ${zScore.toFixed(2)}`,
      });
    }
  }

  return { valid: issues.filter((i) => i.severity === "error").length === 0, issues };
}

function validateTemporalConsistency(
  batch: CreditBatch,
  project: Project
): { valid: boolean; issues: ValidationIssue[] } {
  const issues: ValidationIssue[] = [];

  const startDate = new Date(batch.startDate);
  const endDate = new Date(batch.endDate);
  const issuanceDate = new Date(batch.issuanceDate);

  if (isNaN(startDate.getTime()) || isNaN(endDate.getTime())) {
    issues.push({
      severity: "error",
      component: "temporal",
      message: "Batch start or end date is invalid",
      evidence: `start: ${batch.startDate}, end: ${batch.endDate}`,
    });
    return { valid: false, issues };
  }

  if (endDate <= startDate) {
    issues.push({
      severity: "error",
      component: "temporal",
      message: "Batch end date must be after start date",
      evidence: `start: ${batch.startDate}, end: ${batch.endDate}`,
    });
  }

  if (issuanceDate < endDate) {
    issues.push({
      severity: "warning",
      component: "temporal",
      message: "Issuance date is before monitoring period end -- verify MRV completion",
      evidence: `issuance: ${batch.issuanceDate}, period end: ${batch.endDate}`,
    });
  }

  const periodDays =
    (endDate.getTime() - startDate.getTime()) / (1000 * 60 * 60 * 24);
  if (periodDays > 365 * 10) {
    issues.push({
      severity: "warning",
      component: "temporal",
      message: `Monitoring period (${Math.round(periodDays)} days) exceeds 10 years`,
      evidence: null,
    });
  }

  return {
    valid: issues.filter((i) => i.severity === "error").length === 0,
    issues,
  };
}

async function crossReferenceMRV(
  koiClient: KOIMCPClient,
  batch: CreditBatch,
  project: Project
): Promise<{ valid: boolean; issues: ValidationIssue[] }> {
  const issues: ValidationIssue[] = [];

  try {
    const mrvResults = await koiClient.search({
      query: `MRV report ${project.id} ${batch.startDate} ${batch.endDate}`,
      intent: "technical_howto",
      limit: 10,
    });

    if (!mrvResults.results || mrvResults.results.length === 0) {
      issues.push({
        severity: "warning",
        component: "mrv",
        message: "No MRV report found in KOI for this batch period",
        evidence: `Searched for project ${project.id}, period ${batch.startDate} to ${batch.endDate}`,
      });
      return { valid: false, issues };
    }

    // Check MRV documents reference the correct project and period
    const relevantDocs = mrvResults.results.filter((doc) => {
      const content = (doc.content ?? "").toLowerCase();
      return (
        content.includes(project.id.toLowerCase()) ||
        content.includes(batch.batchDenom.toLowerCase())
      );

    if (relevantDocs.length === 0) {
      issues.push({
        severity: "warning",
        component: "mrv",
        message: "MRV documents found but none reference this specific project or batch",
        evidence: `${mrvResults.results.length} documents found, 0 relevant`,
      });
    }

    return {
      valid: relevantDocs.length > 0,
      issues,
    };
  } catch {
    issues.push({
      severity: "info",
      component: "mrv",
      message: "Could not perform MRV cross-reference (KOI search error)",

    });
    return { valid: false, issues };
  }
```

```typescript
      }
    }

    let recommendation: DecisionOutcome;
    if (score >= DEFAULT_DECISION_THRESHOLDS.proceed) {
      recommendation = "APPROVE";
    } else if (score >= DEFAULT_DECISION_THRESHOLDS.flagForReview) {
      recommendation = "FLAG_FOR_REVIEW";
    } else {
      recommendation = "ESCALATE";
    }

    const shouldContinue = score >= DEFAULT_DECISION_THRESHOLDS.flagForReview;

    return {
      shouldContinue,
      score,
      recommendation,
      missingComponents,
      message: shouldContinue
        ? `Methodology completeness ${(score * 100).toFixed(0)}% -- workflow may proceed.`
        : `Methodology completeness ${(score * 100).toFixed(0)}% is below threshold. ` +
          `Missing: ${missingComponents.join(", ")}. Escalating.`,
```

```typescript
import type { KOIMCPClient } from "@regen/plugin-koi-mcp";

interface DuplicateResult {
  isDuplicate: boolean;
  highestSimilarity: number;
  potentialDuplicates: Array<{
    id: string;
    title: string;
    similarity: number;
    uri: string;

  message: string;
}

export const duplicateDetectorEvaluator: Evaluator = {
  name: "duplicate_detector",
  description:
    "Detects potential duplicate applications or projects using KOI embedding " +
    "similarity. Runs on ANALYZE_METHODOLOGY and VALIDATE_PROJECT results.",

  similes: ["duplicate check", "similarity check"],

  alwaysRun: true,

  validate: async (
    _runtime: IAgentRuntime,
    message: Memory
  ): Promise<boolean> => {
    const action = message.content.action;
    return (
      action === "ANALYZE_METHODOLOGY" ||
      action === "VALIDATE_PROJECT" ||
      action === "REVIEW_BATCH"
    );
  },

  handler: async (
    runtime: IAgentRuntime,
    message: Memory
  ): Promise<DuplicateResult> => {
    const koiClient = runtime.getService<KOIMCPClient>("koi-mcp");

    if (!koiClient) {
      return {
        isDuplicate: false,
        highestSimilarity: 0,
        potentialDuplicates: [],
        confidence: 0,
        message: "KOI client unavailable -- cannot check for duplicates.",
      };
    }

    const metadata = message.content.metadata as Record<string, unknown> | undefined;
    const searchTarget =
      (metadata?.methodologyId as string) ??
      (metadata?.projectId as string) ??
      (metadata?.batchDenom as string) ??
      message.content.text?.substring(0, 200);

    if (!searchTarget) {
      return {
        isDuplicate: false,
        highestSimilarity: 0,
        potentialDuplicates: [],
        confidence: 0,
        message: "No identifiable content to check for duplicates.",
      };
    }

    try {
      const results = await koiClient.search({
        query: searchTarget,
        limit: 15,
      });

      if (!results.results || results.results.length === 0) {
        return {
          isDuplicate: false,
          highestSimilarity: 0,
          potentialDuplicates: [],
          confidence: 0.9,
          message: "No similar entries found in knowledge base.",
        };
      }

      // Use KOI embedding similarity for accurate comparison
      const potentialDuplicates: DuplicateResult["potentialDuplicates"] = [];
      let highestSimilarity = 0;

      for (const result of results.results) {
        let similarity: number;
        try {
          similarity = await koiClient.computeSimilarity(
            searchTarget,
            result.content ?? result.title ?? ""
          );
        } catch {
          // Fall back to search score as proxy
          similarity = result.score ?? 0;
        }

        if (similarity > 0.7) {
          potentialDuplicates.push({
            id: result.id,
            title: result.title ?? "Untitled",
            similarity,
            uri: result.uri ?? "",
          });
        }

        highestSimilarity = Math.max(highestSimilarity, similarity);
      }

      // Sort by similarity descending
      potentialDuplicates.sort((a, b) => b.similarity - a.similarity);

      const isDuplicate = highestSimilarity > 0.85;
      const confidence =
        highestSimilarity > 0.9 ? 0.95 : highestSimilarity > 0.8 ? 0.8 : 0.6;

      return {
        isDuplicate,
        highestSimilarity,
        potentialDuplicates: potentialDuplicates.slice(0, 5),
        confidence,
        message: isDuplicate
          ? `DUPLICATE DETECTED: ${potentialDuplicates[0]?.title} (similarity: ${(highestSimilarity * 100).toFixed(1)}%)`
          : potentialDuplicates.length > 0
            ? `${potentialDuplicates.length} similar entries found (max similarity: ${(highestSimilarity * 100).toFixed(1)}%)`
            : "No duplicates detected.",
      };
    } catch (error) {
      const msg = error instanceof Error ? error.message : String(error);
      return {
        isDuplicate: false,
        highestSimilarity: 0,
        potentialDuplicates: [],
        confidence: 0,
        message: `Duplicate check failed: ${msg}`,
      };
    }
  },
```

```typescript
function classifyProposal(proposal: Proposal): ProposalCategory {
  const type = proposal.content?.["@type"] ?? "";

  if (type.includes("SoftwareUpgrade")) return "software_upgrade";
  if (type.includes("ParameterChange") || type.includes("UpdateParams"))
    return "parameter_change";
  if (type.includes("CommunityPoolSpend")) return "community_pool_spend";
  if (type.includes("Text")) return "text_proposal";
  if (
    type.includes("CreditType") ||
    type.includes("AllowDenom") ||
    type.includes("CreditClass")
  )
    return "credit_type_management";
  return "other";
}

function assessRiskLevel(
  category: ProposalCategory,
  proposal: Proposal
): "low" | "medium" | "high" | "critical" {
  if (category === "software_upgrade") return "high";
  if (category === "parameter_change") {
    const changes = proposal.content?.changes ?? [];
    const governanceChanges = changes.filter(
      (c) => c.subspace === "gov" || c.subspace === "staking"
    );
    return governanceChanges.length > 0 ? "high" : "low";
  }
  if (category === "community_pool_spend") {
    const amount = proposal.content?.amount ?? [];
    const totalUregen = amount.reduce(
      (sum, c) => sum + (c.denom === "uregen" ? BigInt(c.amount) : BigInt(0)),
      BigInt(0)
    );
    const regenAmount = Number(totalUregen) / 10 ** REGEN_DECIMALS;
    if (regenAmount > 100_000) return "high";
    if (regenAmount > 10_000) return "medium";
    return "low";
  }
  return "low";
}

function assessImpact(
  category: ProposalCategory,
  proposal: Proposal
): { technical: string; economic: string; governance: string } {
  const type = proposal.content?.["@type"] ?? "";

  let technical = "Low";
  let economic = "Low";
  let governance = "Low";

  if (type.includes("SoftwareUpgrade")) {
    technical = "High -- Chain upgrade requires coordinated validator restart";
    governance = "Medium -- Sets precedent for upgrade cadence";
  }
  if (type.includes("Parameter")) {
    technical = "Low -- Configuration change, no code deployment";
    if (proposal.content?.changes?.some((c) => c.subspace === "staking")) {
      economic = "Medium -- Staking parameters affect yield";
    }
  }
  if (type.includes("CommunityPoolSpend")) {
    economic = "Direct -- Treasury disbursement";
    governance = "Medium -- Sets spending precedent";
  }
  if (type.includes("AllowDenom") || type.includes("CreditType")) {
    economic = "Medium -- Enables new market trading pairs or credit types";
  }

  return { technical, economic, governance };

function projectOutcome(
  tally: TallyResult,
  totalBonded: bigint,
  timeRemainingHours: number
): { prediction: "pass" | "reject" | "uncertain"; confidence: number; projectedTurnout: number } {
  const yes = BigInt(tally.yes ?? "0");
  const no = BigInt(tally.no ?? "0");
  const abstain = BigInt(tally.abstain ?? "0");
  const veto = BigInt(tally.noWithVeto ?? "0");
  const totalVoted = yes + no + abstain + veto;

  if (totalBonded === BigInt(0)) {
    return { prediction: "uncertain", confidence: 0, projectedTurnout: 0 };
  }

  const turnout = Number(totalVoted) / Number(totalBonded);
  const quorumReached = turnout >= ALERT_THRESHOLDS.QUORUM_REQUIRED;

  const yesPercent = totalVoted > 0 ? Number(yes) / Number(totalVoted) : 0;
  const vetoPercent = totalVoted > 0 ? Number(veto) / Number(totalVoted) : 0;

  if (!quorumReached && timeRemainingHours < 24) {
    return {
      prediction: "reject",
      confidence: 0.7,
      projectedTurnout: turnout,
    };
  }

  if (vetoPercent >= ALERT_THRESHOLDS.VETO_THRESHOLD) {
    return {
      prediction: "reject",
      confidence: 0.9,
      projectedTurnout: turnout,
    };
  }

  if (yesPercent > 0.66 && quorumReached) {
    const confidence = Math.min(0.95, 0.5 + yesPercent * 0.3 + turnout * 0.2);
    return { prediction: "pass", confidence, projectedTurnout: turnout };
  }

  if (yesPercent < 0.4 && turnout > 0.2) {
    return {
      prediction: "reject",
      confidence: 0.75,
      projectedTurnout: turnout,
    };
  }

  return {
    prediction: "uncertain",
    confidence: 0.4,
    projectedTurnout: turnout,
  };
}
```

```typescript
interface PriceDataPoint {
  price: number;
  timestamp: string;
  batchDenom: string;
}

function computeZScore(value: number, mean: number, stdDev: number): number {
  if (stdDev === 0) return 0;
  return (value - mean) / stdDev;
}

function computeStats(values: number[]): { mean: number; stdDev: number } {
  if (values.length === 0) return { mean: 0, stdDev: 0 };

  const mean = values.reduce((sum, v) => sum + v, 0) / values.length;
  const variance =
    values.reduce((sum, v) => sum + Math.pow(v - mean, 2), 0) / values.length;

  return { mean, stdDev: Math.sqrt(variance) };
}

function determineSeverity(
  zScore: number,
  deviationPercent: number
): PriceAnomaly["severity"] {
  if (
    Math.abs(zScore) >= ALERT_THRESHOLDS.Z_SCORE_CRITICAL ||
    Math.abs(deviationPercent) >= ALERT_THRESHOLDS.PRICE_DEVIATION_CRITICAL
  ) {
    return "critical";
  }
  if (
    Math.abs(zScore) >= ALERT_THRESHOLDS.Z_SCORE_ANOMALY ||
    Math.abs(deviationPercent) >= ALERT_THRESHOLDS.PRICE_DEVIATION_ALERT
  ) {
    return "alert";
  }
  if (Math.abs(deviationPercent) >= ALERT_THRESHOLDS.PRICE_DEVIATION_WATCH) {
    return "watch";
  }
  return "normal";
}

function inferPossibleCauses(
  anomaly: PriceAnomaly,
  allOrders: SellOrder[]
): string[] {
  const causes: string[] = [];

  // Check for seller concentration
  const sellerCounts = new Map<string, number>();
  for (const order of allOrders.filter(
    (o) => o.batchDenom === anomaly.batchDenom
  )) {
    sellerCounts.set(
      order.seller,
      (sellerCounts.get(order.seller) ?? 0) + 1
    );
  }

  const totalOrders = allOrders.filter(
    (o) => o.batchDenom === anomaly.batchDenom
  ).length;

  for (const [seller, count] of sellerCounts) {
    if (count / totalOrders > ALERT_THRESHOLDS.SINGLE_ACTOR_VOLUME_WATCH) {
      causes.push(
        `Single seller ${seller.substring(0, 15)}... controls ${((count / totalOrders) * 100).toFixed(0)}% of orders`
      );
    }
  }

  if (anomaly.zScore > 2) {
    causes.push("Price significantly above historical average -- possible supply shortage or demand spike");
  } else if (anomaly.zScore < -2) {
    causes.push("Price significantly below historical average -- possible panic selling or new large supply");
  }

  if (totalOrders < 5) {
    causes.push("Very few active sell orders -- thin market may amplify price movements");
  }

  if (causes.length === 0) {
    causes.push("No specific cause identified -- may reflect normal market dynamics");
  }

  return causes;
}

function formatAnomalyReport(anomalies: PriceAnomaly[]): string {
  if (anomalies.length === 0) {
    return "## Market Anomaly Scan\n\nNo anomalies detected. All credit types trading within normal ranges.";
  }

  const rows = anomalies
    .map(
      (a) =>
        `| ${a.creditType} | ${a.currentPrice.toFixed(2)} | ${a.movingAverage7d.toFixed(2)} | ${a.zScore.toFixed(2)} | ${a.deviationPercent > 0 ? "+" : ""}${a.deviationPercent.toFixed(1)}% | ${a.severity.toUpperCase()} |`
    )
    .join("\n");

  let report = `## Market Anomaly Scan

### Detected Anomalies
| Credit Type | Current Price | 7d Avg | Z-Score | Deviation | Severity |
|-------------|--------------|--------|---------|-----------|----------|
${rows}

### Details`;

  for (const anomaly of anomalies.filter((a) => a.severity !== "normal")) {
    report += `\n\n#### ${anomaly.creditType} (${anomaly.batchDenom})
- **Current**: ${anomaly.currentPrice.toFixed(2)} REGEN
- **7-day avg**: ${anomaly.movingAverage7d.toFixed(2)} REGEN
- **30-day avg**: ${anomaly.movingAverage30d.toFixed(2)} REGEN
- **Z-score**: ${anomaly.zScore.toFixed(2)}
- **Possible causes**: ${anomaly.possibleCauses.join("; ")}`;
  }

  return report;
}
```

```typescript
function computeHerfindahlIndex(shares: number[]): number {
  return shares.reduce((sum, s) => sum + s * s, 0);
}

function assessHealth(
  totalOrders: number,
  depthScore: number,
  concentrationIndex: number,
  bidAskSpread: number
): { score: number; rating: LiquidityAssessment["healthRating"] } {
  let score = 0;

  // Order count component (0-25)
  score += Math.min(25, totalOrders * 2.5);

  // Depth component (0-25)
  score += depthScore * 25;

  // Concentration component (0-25) -- lower is better
  score += (1 - concentrationIndex) * 25;

  // Spread component (0-25) -- lower is better
  score += Math.max(0, 25 - bidAskSpread * 100);

  const normalized = score / 100;

  let rating: LiquidityAssessment["healthRating"];
  if (normalized >= 0.75) rating = "healthy";
  else if (normalized >= 0.5) rating = "adequate";
  else if (normalized >= 0.25) rating = "thin";
  else rating = "critical";

  return { score: normalized, rating };
}

function formatLiquidityReport(assessments: LiquidityAssessment[]): string {
  if (assessments.length === 0) {
    return "## Liquidity Assessment\n\nNo active markets to assess.";
  }

  const summaryRows = assessments
    .map(
      (a) =>
        `| ${a.creditType} | ${a.totalSellOrders} | ${a.totalVolume.toLocaleString(undefined, { maximumFractionDigits: 0 })} | ${(a.bidAskSpread * 100).toFixed(1)}% | ${(a.healthScore * 100).toFixed(0)}/100 | ${a.healthRating.toUpperCase()} |`
    )
    .join("\n");

  let report = `## Liquidity Assessment

### Market Overview
| Credit Type | Orders | Volume | Spread | Health Score | Rating |
|-------------|--------|--------|--------|-------------|--------|
${summaryRows}`;

  // Detail any critical or thin markets
  const concerns = assessments.filter(
    (a) => a.healthRating === "critical" || a.healthRating === "thin"
  );

  if (concerns.length > 0) {
    report += `\n\n### Liquidity Concerns`;
    for (const c of concerns) {
      report += `\n\n#### ${c.creditType} -- ${c.healthRating.toUpperCase()}
- Sell orders: ${c.totalSellOrders}
- Concentration index (HHI): ${(c.concentrationIndex * 10000).toFixed(0)} (${c.concentrationIndex > 0.25 ? "HIGH" : "moderate"})
```

```typescript
    });
  }

  // Volume concentration signal
  const totalVolume = retirements.reduce(
    (sum, r) => sum + parseFloat(r.amount || "0"),
    0
  );

    const ownerVolume = retirements
      .filter((r) => r.owner === owner)
      .reduce((sum, r) => sum + parseFloat(r.amount || "0"), 0);
    return Math.max(maxVol, ownerVolume);
  }, 0);

  if (totalVolume > 0 && topRetirerVolume / totalVolume > 0.5) {
    signals.push({
      signal: "Concentrated retirement demand",
      strength: "strong",
      evidence: `Single entity accounts for ${((topRetirerVolume / totalVolume) * 100).toFixed(0)}% of retirement volume`,
    });
  }

  // Reason-based signals
  const reasons = retirements
    .map((r) => r.reason?.toLowerCase() ?? "")

    (r) =>

  );

      strength: complianceReasons.length > 5 ? "strong" : "weak",

    });

      r.includes("voluntary") ||

  );

function determineTrend(retirements: Retirement[]): "increasing" | "stable" | "decreasing" {
  if (retirements.length < 5) return "stable";

  // Split into halves and compare volume
  const mid = Math.floor(retirements.length / 2);
  const firstHalf = retirements.slice(0, mid);
  const secondHalf = retirements.slice(mid);

  const firstVolume = firstHalf.reduce(
    (sum, r) => sum + parseFloat(r.amount || "0"),
    0
  );
  const secondVolume = secondHalf.reduce(
    (sum, r) => sum + parseFloat(r.amount || "0"),
    0
  );

  const changeRatio = firstVolume > 0 ? secondVolume / firstVolume : 1;

  if (changeRatio > 1.2) return "increasing";
  if (changeRatio < 0.8) return "decreasing";
  return "stable";
}
```

```typescript
function extractValidatorRef(text: string): string | null {
  // Match regenvaloper addresses or moniker names
  const addrMatch = text.match(/regenvaloper[a-z0-9]{39}/i);
  if (addrMatch) return addrMatch[0];

function computeUptimeScore(
  missedBlocks: number,
  windowSize: number = 10_000
): number {
  const signedBlocks = windowSize - missedBlocks;
  return Math.max(0, Math.min(1, signedBlocks / windowSize));
}

function computeCommissionScore(rate: string): number {
  const commission = parseFloat(rate);
  if (isNaN(commission)) return 0.5;

  // Ideal range: 5-10%. Penalize extremes.
  if (commission >= 0.05 && commission <= 0.10) return 1.0;
  if (commission < 0.01) return 0.3; // Unsustainably low
  if (commission <= 0.15) return 0.8;
  if (commission <= 0.20) return 0.5;
  return 0.2; // > 20%
}

function computeSelfDelegationScore(
  selfDelegation: number,
  totalTokens: number
): number {
  if (totalTokens === 0) return 0;
  const ratio = selfDelegation / totalTokens;

  // Higher self-delegation = more skin in the game
  if (ratio >= 0.1) return 1.0;
  if (ratio >= 0.05) return 0.8;
  if (ratio >= 0.01) return 0.5;
  return 0.2;
}

function detectAlerts(
  validator: Validator,
  missedBlocks: number,
  uptimeScore: number
): ValidatorAlert[] {
  const alerts: ValidatorAlert[] = [];
  const now = new Date().toISOString();

  if (validator.jailed) {
    alerts.push({
      type: "jailing",
      severity: "critical",
      message: `Validator ${validator.description.moniker} is JAILED`,
      timestamp: now,
    });
  }

  if (missedBlocks > ALERT_THRESHOLDS.MISSED_BLOCKS_WARNING) {
    alerts.push({
      type: "missed_blocks",
      severity: missedBlocks > 500 ? "high" : "medium",
      message: `${missedBlocks} blocks missed in current signing window`,
      timestamp: now,
    });
  }

  if (uptimeScore < 0.95) {
    alerts.push({
      type: "downtime",
      severity: uptimeScore < 0.9 ? "high" : "medium",
      message: `Uptime ${(uptimeScore * 100).toFixed(1)}% is below 95% threshold`,
      timestamp: now,
    });
  }

  const commission = parseFloat(
    validator.commission.commissionRates.rate || "0"
  );
  if (commission > 0.2) {
    alerts.push({
      type: "commission_change",
      severity: "low",
      message: `Commission rate ${(commission * 100).toFixed(1)}% is above 20%`,
      timestamp: now,
    });
  }

  if (validator.status === "BOND_STATUS_UNBONDING") {
    alerts.push({
      type: "unbonding",
      severity: "high",
      message: `Validator is unbonding (height: ${validator.unbondingHeight})`,
      timestamp: now,
    });
  }

  return alerts;
}
```

```typescript
function computeNakamotoCoefficient(stakes: number[]): number {
  if (stakes.length === 0) return 0;

  const sorted = [...stakes].sort((a, b) => b - a);
  const totalStake = sorted.reduce((sum, s) => sum + s, 0);
  if (totalStake === 0) return 0;

  let cumulativeStake = 0;
  let count = 0;

  for (const stake of sorted) {
    cumulativeStake += stake;
    count++;
    if (cumulativeStake / totalStake > 1 / 3) {
      return count;
    }
  }

  return stakes.length;
}

function computeGiniCoefficient(values: number[]): number {
  if (values.length === 0) return 0;

  const sorted = [...values].sort((a, b) => a - b);
  const n = sorted.length;
  const mean = sorted.reduce((sum, v) => sum + v, 0) / n;

  if (mean === 0) return 0;

  let sumAbsDiff = 0;
  for (let i = 0; i < n; i++) {
    for (let j = 0; j < n; j++) {
      sumAbsDiff += Math.abs(sorted[i]! - sorted[j]!);
    }
  }

  return sumAbsDiff / (2 * n * n * mean);
}

function computeHerfindahlIndex(shares: number[]): number {
  return shares.reduce((sum, s) => sum + s * s, 0);
}

function estimateGeographicDiversity(validators: Validator[]): number {
  // Heuristic: use identity field to estimate different organizations
  const identities = new Set(
    validators
      .map((v) => v.description.identity)
      .filter((id) => id.length > 0)
  );
  const websites = new Set(
    validators
      .map((v) => {
        try {
          return new URL(v.description.website).hostname;
        } catch {
          return "";
        }
      })
      .filter((h) => h.length > 0)
  );

  // Normalize: more unique identities/websites = more diversity
  const identityScore = Math.min(1.0, identities.size / 30);
  const websiteScore = Math.min(1.0, websites.size / 30);

  return (identityScore + websiteScore) / 2;
}

      const bondedRes = await ledgerClient.listValidators({
        status: "BOND_STATUS_BONDED",
        limit: 200,
      });
      bondedValidators = bondedRes.items ?? [];

      // Also fetch unbonded/jailed for complete picture
      const allRes = await ledgerClient.listValidators({
        limit: 300,
      });
      allValidators = allRes.items ?? [];
    } catch (error) {
      const msg = error instanceof Error ? error.message : String(error);
      callback({ text: `Failed to fetch validator set: ${msg}` });
      return false;
    }

    if (bondedValidators.length === 0) {
      callback({ text: "No bonded validators found." });
      return false;
    }

    // Compute stakes in REGEN
    const stakes = bondedValidators.map(
      (v) => Number(BigInt(v.tokens) / BigInt(10 ** REGEN_DECIMALS))
    );
    const totalStake = stakes.reduce((sum, s) => sum + s, 0);
    const stakeShares = stakes.map((s) => (totalStake > 0 ? s / totalStake : 0));

    // Sorted descending for top-N calculations
    const sortedShares = [...stakeShares].sort((a, b) => b - a);

    const nakamotoCoefficient = computeNakamotoCoefficient(stakes);
    const giniCoefficient = computeGiniCoefficient(stakes);
    const herfindahlIndex = computeHerfindahlIndex(stakeShares);

    const top5StakePercent =
      sortedShares.slice(0, 5).reduce((sum, s) => sum + s, 0) * 100;
    const top10StakePercent =
      sortedShares.slice(0, 10).reduce((sum, s) => sum + s, 0) * 100;

    const activeValidatorCount = bondedValidators.length;
    const jailedValidatorCount = allValidators.filter(
      (v) => v.jailed
    ).length;
    const geographicDiversity = estimateGeographicDiversity(bondedValidators);

    // Risk level
    let riskLevel: DecentralizationMetrics["riskLevel"];
    if (
      nakamotoCoefficient < ALERT_THRESHOLDS.NAKAMOTO_COEFFICIENT_CRITICAL ||
      top5StakePercent > 50
    ) {
      riskLevel = "critical";
    } else if (
      nakamotoCoefficient < 10 ||
      top5StakePercent > ALERT_THRESHOLDS.TOP_5_STAKE_CONCERN * 100
    ) {
      riskLevel = "concerning";
    } else if (giniCoefficient > 0.7) {
      riskLevel = "adequate";
    } else {
      riskLevel = "healthy";
    }

    // Recommendations
    const recommendations: string[] = [];

    if (nakamotoCoefficient < ALERT_THRESHOLDS.NAKAMOTO_COEFFICIENT_CRITICAL) {
      recommendations.push(
        `Nakamoto coefficient (${nakamotoCoefficient}) is below ${ALERT_THRESHOLDS.NAKAMOTO_COEFFICIENT_CRITICAL}. Consider community campaigns to redistribute stake to smaller validators.`
      );
    }

    if (top5StakePercent > 40) {
      recommendations.push(
        `Top 5 validators control ${top5StakePercent.toFixed(1)}% of stake. Encourage delegators to diversify across the validator set.`
      );
    }

    if (giniCoefficient > 0.7) {
      recommendations.push(
        `High stake inequality (Gini: ${giniCoefficient.toFixed(3)}). Consider foundation delegation programs for smaller validators.`
      );
    }

    if (geographicDiversity < 0.3) {
      recommendations.push(
        "Low geographic/organizational diversity. Encourage validators from underrepresented regions."
      );
    }

    if (jailedValidatorCount > 5) {
      recommendations.push(
        `${jailedValidatorCount} jailed validators. Review jailing conditions and validator support resources.`
      );
    }

    const metrics: DecentralizationMetrics = {
      nakamotoCoefficient,
      giniCoefficient,
      herfindahlIndex,
      top5StakePercent,
      top10StakePercent,
      activeValidatorCount,
      jailedValidatorCount,
      geographicDiversity,
      riskLevel,
      recommendations,
    };

    const response = formatDecentralizationReport(metrics);
    callback({ text: response });

    await runtime.messageManager.createMemory({
      id: crypto.randomUUID(),
      agentId: runtime.agentId,
      userId: message.userId,
      roomId: message.roomId,
      content: {
        text: response,
        action: "ASSESS_DECENTRALIZATION",
        metadata: metrics,
      },
    });

    return true;
  },
};
```

```typescript
// packages/plugin-regen-validator/src/evaluators/performance-threshold.ts

import type { Evaluator, IAgentRuntime, Memory } from "@elizaos/core";
import type {
  ValidatorAssessment,
  DecentralizationMetrics,
  InterAgentMessage,
} from "@regen/shared";
import { ALERT_THRESHOLDS } from "@regen/shared";

interface PerformanceEvalResult {
  shouldContinue: boolean;
  alertLevel: "none" | "warning" | "critical";
  validatorsAtRisk: number;
  decentralizationConcern: boolean;
  message: string;
  escalations: InterAgentMessage[];
}

export const performanceThresholdEvaluator: Evaluator = {
  name: "performance_threshold",
  description:
    "Evaluates validator assessment and decentralization results against " +
    "thresholds and generates inter-agent alerts for critical issues.",

  similes: ["validator alert", "performance check"],

  alwaysRun: false,

  validate: async (
    _runtime: IAgentRuntime,
    message: Memory
  ): Promise<boolean> => {
    const action = message.content.action;
    return (
      action === "ASSESS_VALIDATOR" || action === "ASSESS_DECENTRALIZATION"
    );
  },

  handler: async (
    runtime: IAgentRuntime,
    message: Memory
  ): Promise<PerformanceEvalResult> => {
    const metadata = message.content.metadata as
      | ValidatorAssessment
      | DecentralizationMetrics
      | undefined;

    if (!metadata) {
      return {
        shouldContinue: true,
        alertLevel: "none",
        validatorsAtRisk: 0,
        decentralizationConcern: false,
        message: "No assessment data to evaluate.",
        escalations: [],
      };
    }

    const escalations: InterAgentMessage[] = [];

    // Handle validator assessment
    if ("compositeScore" in metadata) {
      const assessment = metadata as ValidatorAssessment;
      const isCritical = assessment.riskLevel === "critical";
      const isHighRisk = assessment.riskLevel === "high";

      if (isCritical || isHighRisk) {
        escalations.push({
          id: crypto.randomUUID(),
          from: "AGENT-004-VALIDATOR",
          to: ["AGENT-002-GOVERNANCE"],
          priority: isCritical ? "urgent" : "high",
          type: "alert",
          payload: {
            alertType: "validator_performance_degraded",
            validatorAddress: assessment.operatorAddress,
            moniker: assessment.moniker,
            compositeScore: assessment.compositeScore,
            riskLevel: assessment.riskLevel,
            alerts: assessment.alerts,
          },
          correlationId: null,
          timestamp: new Date().toISOString(),
          expiresAt: new Date(
            Date.now() + 24 * 60 * 60 * 1000
          ).toISOString(),
        });
      }

      return {
        shouldContinue: true,
        alertLevel: isCritical ? "critical" : isHighRisk ? "warning" : "none",
        validatorsAtRisk: isCritical || isHighRisk ? 1 : 0,
        decentralizationConcern: false,
        message: isCritical
          ? `CRITICAL: Validator ${assessment.moniker} has critical risk level (score: ${(assessment.compositeScore * 100).toFixed(1)}%). Alert dispatched.`
          : isHighRisk
            ? `WARNING: Validator ${assessment.moniker} has high risk level (score: ${(assessment.compositeScore * 100).toFixed(1)}%).`
            : `Validator ${assessment.moniker} performance within acceptable thresholds.`,
        escalations,
      };
    }

    // Handle decentralization assessment
    if ("nakamotoCoefficient" in metadata) {
      const decMetrics = metadata as DecentralizationMetrics;
      const isCritical = decMetrics.riskLevel === "critical";
      const isConcerning = decMetrics.riskLevel === "concerning";

      if (isCritical) {
        escalations.push({
          id: crypto.randomUUID(),
          from: "AGENT-004-VALIDATOR",
          to: [
            "AGENT-002-GOVERNANCE",
            "AGENT-003-MARKET",
          ],
          priority: "urgent",
          type: "alert",
          payload: {
            alertType: "decentralization_critical",
            nakamotoCoefficient: decMetrics.nakamotoCoefficient,
            giniCoefficient: decMetrics.giniCoefficient,
            top5StakePercent: decMetrics.top5StakePercent,
            recommendations: decMetrics.recommendations,
          },
          correlationId: null,
          timestamp: new Date().toISOString(),
          expiresAt: new Date(
            Date.now() + 7 * 24 * 60 * 60 * 1000
          ).toISOString(),
        });
      }

      return {
        shouldContinue: true,
        alertLevel: isCritical ? "critical" : isConcerning ? "warning" : "none",
        validatorsAtRisk: 0,
        decentralizationConcern: isCritical || isConcerning,
        message: isCritical
          ? `CRITICAL: Network decentralization at critical level. Nakamoto coefficient: ${decMetrics.nakamotoCoefficient}. Alert dispatched.`
          : isConcerning
            ? `WARNING: Decentralization metrics show concerning trends. Nakamoto coefficient: ${decMetrics.nakamotoCoefficient}.`
            : `Network decentralization metrics within healthy range. Nakamoto coefficient: ${decMetrics.nakamotoCoefficient}.`,
        escalations,
      };
    }

    return {
      shouldContinue: true,
      alertLevel: "none",
      validatorsAtRisk: 0,
      decentralizationConcern: false,
      message: "Unknown assessment type.",
      escalations: [],
    };
  },
};
```

This reference implementation provides **production-grade TypeScript** for the entire Regen Network agent system:

|--------|-------|---------|------------|------------------|
| `plugin-regen-ledger` | Shared | -- | `chainHealthEvaluator` | MCP client, state provider |
| `plugin-regen-koi` | Shared | -- | -- | Knowledge graph access |

| `plugin-regen-validator` | AGENT-004 | `assessValidator`, `analyzeDelegation`, `assessDecentralization` | `performanceThreshold` | Nakamoto/Gini/HHI, composite scoring |

### Key Design Decisions

1. **OODA Loop Pattern**: Every agent action follows Observe-Orient-Decide-Act. The observe phase gathers data from MCP servers; orient cross-references with KOI knowledge; decide computes confidence scores; act formats output and stores in memory.

2. **Confidence-Based Escalation**: Configurable thresholds (`DEFAULT_DECISION_THRESHOLDS`) control when agents proceed autonomously versus flagging for human review. This is the primary safety mechanism.

3. **Inter-Agent Messaging**: Evaluators generate `InterAgentMessage` objects when critical conditions are detected (price anomalies, validator failures, decentralization concerns). These messages use typed payloads and priority levels.

4. **Memory Persistence**: Every action stores its output in ElizaOS memory with structured metadata. This enables post-vote prediction accuracy tracking (AGENT-002) and historical comparison (all agents).

5. **Graceful Degradation**: All MCP calls use retry logic with exponential backoff. Provider and evaluator failures are non-fatal -- agents log errors and continue with available data.
