## Quick Navigation

### For Humans

Start with the main [CONTRIBUTING.md](../../CONTRIBUTING.md) for an overview, then explore:
- Phase-specific documentation in `/phase-1/`, `/phase-2/`, `/phase-3/`
- Governance context in `/docs/governance/`
- Integration guides in `/docs/integration/`

### For Agents

Agents should:
1. Review [CONTRIBUTING.md](../../CONTRIBUTING.md) for boundaries and protocols
2. Understand [skill-organization.md](./skill-organization.md) for capability structure
3. Follow the agent commit format for all contributions
4. Flag items requiring human review
