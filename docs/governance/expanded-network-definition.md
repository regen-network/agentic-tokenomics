# Expanded Regen Network Definition

## From Blockchain to Regenerative Coordination

The term "Regen Network" has evolved beyond its original meaning:

### Original Definition (2017-2020)
> Regen Network: A blockchain for ecological assets and claims

### Expanded Definition (2024+)
> Regen Network: A regenerative coordination infrastructure spanning blockchain, knowledge systems, governance frameworks, and human networks aligned around accelerating ecological regeneration

## Registry and Ledger: Distinct Roles

### Regen Ledger

**Purpose**: Immutable, decentralized state machine for ecological assets

**Responsibilities**:
- Token economics and governance
- Credit lifecycle management (x/ecocredit)
- Data anchoring and attestation (x/data)
- Network consensus and security

### Regen Registry

**Purpose**: Quality assurance and methodology governance layer

**Responsibilities**:
- Credit class standards and approvals
- Methodology development and maintenance
- Verification and validation processes
- Claims engine operation

### Design Philosophy

> "The design of Regen Ledger has been to be registry agnostic, meaning it's meant to be an open registry system where different communities can program in their values, methodologies and standards."
